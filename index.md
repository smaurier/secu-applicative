# 🔐 Sécurité Applicative

Bienvenue dans le module **Sécurité Applicative** du parcours fullstack.

## Pourquoi ce module ?

La sécurité n'est pas une fonctionnalité — c'est une propriété fondamentale de toute application de qualité. Ce module couvre les menaces les plus courantes et les défenses modernes pour les applications web JavaScript/TypeScript.

## Les trois phases

### 🟢 Phase 1 — Fondamentaux
Comprendre le paysage des menaces (OWASP Top 10), les attaques par injection, et la gestion sécurisée de l'authentification.

### 🟡 Phase 2 — Défenses avancées
Implémenter des systèmes d'autorisation robustes, utiliser la cryptographie correctement, et configurer les en-têtes HTTP de sécurité.

### 🔴 Phase 3 — Sécurité système
Sécuriser les APIs, auditer la supply chain, protéger l'infrastructure, et mener des audits de sécurité.

## Comment utiliser ce module

1. **Lire le chapitre** dans `modules/`
2. **Réaliser le lab** correspondant dans `labs/`
3. **Valider avec le quiz** dans `quizzes/`
4. **Consulter le glossaire** pour les termes techniques

## Prérequis

- JavaScript/TypeScript maîtrisés
- HTTP et APIs REST
- Node.js (Express ou NestJS)
- Bases SQL
